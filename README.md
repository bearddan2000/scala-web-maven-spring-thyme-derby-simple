# scala-web-maven-spring-thyme-derby-simple

## Description
A thyme springboot scala maven build,
that connects to in memory derby database.

## Tech stack
- springboot
  - thymeleaf
- maven
  - derby drivers
  - lombok
  - PostConstruct annotation
- bootstrap
- jquery
- dataTable

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`
Available http://localhost

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
